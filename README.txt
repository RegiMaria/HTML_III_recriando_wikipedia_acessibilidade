**Objetivo**

**Curso HTML -Módulo III "Recriando uma página da Wikipédia"** 

Este desafio consiste em treinar a composição da estrutura do site baseado na wikipedia;também aplicar conhecimentos adquiridos sobre semântica e acessibilidade.
Segundo a W3C, a relação entre texto e o fundo adquire sua harmonia, para aqueles cuja visão é comprometida, quando o contraste se ergue para além de 4.5:1.

Para o encerramento do Módulo III do curso HTML, recriamos uma página da wikipedia, pessoalmente e propositalmente escolhi contraste com taxas superior a 14:1.
Além do contraste de cores, adicinonamos rótulos, elementos semânticos, textos alternativos para as imagens e botões de açoes claros. 
Testamos a acessibilidade do site com a plataforma NVDA para a leitura de tela, um programa  que “ler” o site e facilita a inclusão digital de deficientes visuais.
